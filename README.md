# Bahai Online Resource Library

An Android application for offline searching and reading of Bahai documents.

## Features
- Offline document search with full-text capabilities
- Comprehensive collection of official Bahai texts
- Digital note-taking with document references  
- Favorites and bookmarking system
- Clean Material Design interface
- Online resources integration

## Version
Current Version: 0.1.0

## Development Setup
This project requires:
- Android Studio with API 21+ support
- Java/Kotlin support
- Gradle build system

## Document Collections
- `documents/confirmed-official/` - Documents from official Bahai websites
- `documents/pending-permissions/` - Documents requiring permission verification
